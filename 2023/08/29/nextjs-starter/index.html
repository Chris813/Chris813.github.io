快速入门

创建Next.js应用

使用CNA脚手架

npx create-next-app@latest

页面间导航

使用<Link>包裹<a>tag

  • 默认页面路由：‘/’
  • 其他组件路由：组件相对路径

特点：

  • 客户端导航。意味着使用JavaScript进行页面转换，这比浏览器进行的默认导航更快。

客户端导航是指在前端应用程序中，通过浏览器或应用程序的客户端执行导航操作，例如点击链接或按钮，以切换视图、页面或路由，而不需要从服务器请求新的页面。

  • 代码分割及预加载

使用体验：快！不需要引入子组件，不需要注册路由，只需要填一个页面地址

🤔我在实践中包裹a标签报错，需要删除才能运行

原因：从Next.js13开始只需要使用就行。

解决：命令行运行npx @next/codemod new-link .

静态资源

  • Assets

    public下存放静态资源

    对于assets直接引用public下的相对路径

  • 元数据

    在<Head>组件内修改<head>标签内的数据

  • CSS样式

    • Next.js内置了对styled-jsx的支持
    • 对CSS Modules和Sass的内置支持

使用CSS Module

CSS Modules 是一种用于管理和组织 CSS 样式的技术，它旨在解决 CSS 的全局作用域问题，使样式更加模块化和可维护

  • 创建xxx.module.css文件
  • 在组件内引入styles
  • 为标签的classname赋值
